“宇宙最强” IDE —— Web IDE

发表于2024-06-08|更新于2024-06-08|点评
|阅读量:

先说几句：

Web IDE 的作者：bzy（QQ: 204490258），是我入圈多年以来最让我无语的一个人（
熟悉我的人都知道我是 ModPE 转 Android 圈，不过在这期间我也学习了很多新东西（像 nodejs，docker 等等）
我从没有用博客写过某个人，但本次主人公 bzy 我真的忍不了

这人做了啥？

这人可能是对前端太执着了，说出过很多逆天言语，这里我也放了几张图：

首先他自己口口声声称自己是前端，连 Vue 都没用过（甚至这里他还用的是 Vue2），软件叫 WebIDE，结果没有环境只有补全，真的极其可笑。
除此之外，他还有更逆天的操作：

这里帮大家总结了几点：《JS 原生开发香》、《css > 安卓》、《Vue 听都没听过说自己是前端》、《以为前端行业只招原生开发者，Vue 什么都不是》、《jQuery 用的多》
这人怎么说，要是在几年前还能吹吹，但是现在都 2024 年了，谁不知道前端框架里 Vue 是必学的？jQuery 国内用的已经没以前多了，这人只能说是活在过去

《没有 Java 源码是解密不了的》、《首创 js 和 html 加密》，真的无敌了！我这里也没乱说，放几张他打包后软件反编译的 Java 代码看看

可以看到，全是 @JavaScriptInterface 注解，使用过 WebView 的朋友应该知道，这个注解是为了向 JS 映射一个 Java 方法或者属性的，他这里是毫无技术含量，反倒也能声称为原创，真的是脸皮厚。对于 js、html 加密，早在之前 AutoJS 就有，甚至之后的 AutoJS Pro 的快照加密。这也能算原创的话，我写个 123 不也能说我原创了数字？真的不如 AutoJS Pro，以前我还对他们有一点仇恨，直到遇到了 Web IDE 作者才让我明白：世界上怎么有如此厚颜无耻之人！

这张图不想评价了，下一张吧

这张大家当个乐呵就好，足以证明群里大家都对这人很有意见，要不然这位群友也不会这么说

这张真的太好笑了，前面刚说怎么都是 html 开发，结果自己转手整了个 Android 弹窗（乐），也不排除是这家伙对 WebView 弹窗处理进行了自定义（亦或者没改）
到这里知道了为什么群里人都讨厌这家伙，自己没什么技术就在那里原创，不懂装懂还天天说前端比 Android 好的，就像这样：

总座高见！

我有话说：

从这里可以看出，计算机圈不乏还有着许许多多的 “张浩洋”，“张浩洋” 的名言是 —— “我会 Jvav”
这位 Web IDE 作者，我也愿称之为下一代 “张浩洋”，而它的 Web IDE，我也愿称为 “Jvav Next Gen”！
面对这种计算机圈的乱象，我们一定要扼杀在摇篮里，一但冒头必定后患无穷，正如前阵子闹得沸沸扬扬的 CEC-IDE，堪称 “国产之光”
以上就是我所说的，感谢观看
