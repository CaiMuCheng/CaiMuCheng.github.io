先说几句

编程界用两个单词表达对新生事物的兴奋与赞美，那就是 “Hello, world”。
首先感谢 Hexo 提供的静态博客搭建，以及 jerryc127 提供的 Butterfly 主题。

为什么创建博客

主要是两个原因：
1.可以自由地写一写自己的感受、经验、编程技巧，在以后忘记了还可以回来看看。
2.因为认识的大大们都有自己的博客，一开始感觉好厉害就想自己也搭建一个，主要还是能表现自己实际上想的是什么吧（

本博客的历程

目前的这个博客是重建另外美化的，因为之前的博客源码给整丢了（纯纯懒）
本博客最开始搭建于 2022-02-17 22:38:5，那个时候我还是一个在上网课的初二生（纯开摆），今年（2023）准高一。
说起来比较巧的是，Rosemoe 大大在高一的时候搭建了自己的博客，我重新搭建居然也在这一年（

我的经历

我是从小学五六年级开始学编程的，也就是 2018 / 2019 年的时候。
在 2018 年年末，回老家的时候在车上没事干，偶然发现了一个可以制作《我的世界》模组的软件，叫 《JsIDE》，然后跟着里面的教程一步步来，开始走上编程这条路。
在 2019 疫情爆发的时候，人在老家没事情干，就想着根据之前学习的来写个模组，当时我写的用户界面都是用里面的用户界面编辑器写的（因为太菜了），后面是写了最初的作品，叫《苦小怕 JS》。虽然界面简陋，但是这是我人生当中第一个自己亲手写出来的 JS。
应该是同一年，FMP Helper 发布了，它可以直接在网易 MC 中加载 JS 模组，于是我也想做 JS 来赚些小钱。后面发布了 Your Hack 系列，最后是在 7.0.0 版本了终止更新。在后面是 Color Hack（这个是真黑历史，我本来不想列出来的，你绝对不想用它，UI 可以说是伤眼睛）。后面 AndLua 圈兴起，我开始学习 AndLua，写出了另外一个作品，叫 Your Hack·Looper。这个时候我的 Android 开发经验已经很丰富了。
再后面，是 FBlue（现已实锤，就是飞鹰） 回归，把整个圈子圈钱的 JS 都给破解了，其实更多破解的是抄袭二改的。
忘记说了，在 FMP Helper 发布同一年，网易 MC 客户端诞生了，那个时候是创造背包，可以生存直接获取基岩（后面是杀戮光环），JS 圈遭受了一些打击。
再往后是 FMP Helper 宣布停更，JS 圈都在整启动器，这个时候 JS 基本没得救了，不过还是有一些人整出第三方启动器的，比如（BlockLauncher CN），但是它只提供内部 API，与原本的 ModPE API 根本不兼容。
忘记是 2020 还是 2021 年过年的时候，我整出了 Amod，也可以叫内置 JS。原理就是通过改端加修改 Dex，在 MainActivity.onCreate 里面初始化我的代码，然后通过 Rhino 引擎加载 JS。不过也需要调用 JNI，这里指的是 InlineHook。
再后面我就退圈了，可以说令人感慨。也就是今年，我写出了 MuCuteJS，这也是我所写的 JS 中 UI 最好看的一款，也算是弥补一下以前的遗憾吧。
也就是在我退圈之后不久买了电脑，正式开始 Android 开发，从最开始的 Java 到 Kotlin，这期间都不到半年。

结语

也许我以后和残页、Rosemoe 这些大大一样，从事着自己喜欢的职业，贡献着自己的开源项目。
为韶华赋诗一曲，名之青春。

待到秋来九月八，我花开后百花杀。
冲天香阵透长安，满城尽带黄金甲。

唐-黄巢 -《不第后赋菊》
